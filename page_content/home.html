Welcome to Library and More!

Click the library picture to go to the library page

Click the cloud picture to go to the weather page

Click the camera picture to take a selfie!

Or use the navbar to move around



This application has 3 main features

The first one is displaying all chicago libraries in a map. You can click on a marker and it will display information about it
The second is, a weather station. You type in the name of a CITY and then it will display the current real life time weather of it!
The third is if you want to take a selfie or picture! and then display it
